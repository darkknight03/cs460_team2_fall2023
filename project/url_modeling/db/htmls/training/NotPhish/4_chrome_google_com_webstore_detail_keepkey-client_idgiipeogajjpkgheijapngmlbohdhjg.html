Overview
A simple cryptocurrency wallet that integrates with the KeepKey device
With Google deprecating Chrome Apps, the Google Chrome KeepKey client is also now deprecated.  

The official software interface for KeepKey is now via the ShapeShift Platform which can be found going to ShapeShift.com.  There is no account needed to view your balances, send/receive any assets that are supported.

IMPORTANT: Any software found in the google playstore claiming to be the official software interface for KeepKey is a fake one.  Do not use those or risk losing your funds.
