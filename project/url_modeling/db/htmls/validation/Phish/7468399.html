AppConnect

How does it work?

AppConnect connects web applications to supported mobile wallets session is started by a scanning a QR code (desktop) or by clicking an application dapp link (mobile).

What is OneAppConnect?

OneAppConnect is an open source protocol for connecting decentralised applications to mobile wallets with QR code scanning or deep linking. A user can interact securely with any Dapp from their mobile phone, making wallets a safer choice compared to desktop or browser extension wallets.
