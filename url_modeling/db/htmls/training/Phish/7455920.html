Activación de Soft Token
El Soft Token genera claves de seguridad de 6 dígitos que cambian constantemente de forma automática y nunca se repiten. Esta clave de seguridad será requerida para realizar tus transacciones de forma simple y segura.
Elige Confirmar para activarlo y un código será enviado a tu correo el cual debes ingresarlo desde la opción ‘Soft Token BISA’ de la aplicación móvil.

Elige Cancelar para activarlo en otro momento.

¡Perfecto! Ahora continuemos el proceso de activación
1. Revisa tu correo, en el cual encontrarás un código de activación.
2. Toma tu dispositivo móvil y abre o instala la app "e-Bisa+". ¿Necesitas ayuda?
3. Selecciona el menú principal (arriba a la izquierda de la pantalla) y ve a la opción "Soft Token BISA". Ingresa el código de activación para generar el token.
4. Presiona "Continuar" y ve a la pantalla siguiente para ingresar el token generado en la app.
