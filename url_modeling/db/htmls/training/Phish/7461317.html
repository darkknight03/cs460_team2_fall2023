Daapsrevitalization


Open protocol for connecting Wallets to Dapps



What is Daapsrevitalization?

Daapsrevitalization is an open source protocol for connecting decentralised applications to mobile wallets with deep linking. A user can interact securely with any Dapp from their mobile phone, making Daapsrevitalization wallets a safer choice compared to desktop or browser extension wallets.


How does it work?

Daapsrevitalization connects web applications to supported mobile wallets. Daapsrevitalization session is started by clicking an application deep link after linking your wallet.
